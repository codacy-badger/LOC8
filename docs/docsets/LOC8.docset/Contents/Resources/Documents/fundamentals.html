Measurement Units

This project follow SI (The International System of Units)

  • Meter for length.
  • Kilogram for mass.
  • Second for time.
  • Radian for angels.
  • Tesla for magnetic field. (In SI base units: kg⋅s⁻²⋅A⁻¹)
