Quaternion

public struct Quaternion: CustomStringConvertible

Quaternion

Discussion:

Type represents a quaternion (one way of parameterizing rotation). If quaternion is an instance of Quaternion, mathematically it represents the following quaternion:

p = xi + yj + zk + w

Where:

i² = j² = k² = ijk = −1
