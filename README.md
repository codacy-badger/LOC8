# LOC8 [![Build Status](https://travis-ci.com/Marwan-Al-Masri/LOC8.svg?token=H9CZx6r8wEAXyxtfxAbz&branch=master)](https://travis-ci.com/Marwan-Al-Masri/LOC8) ![Documentation](./docs/badge.svg)

## Requirements
- Swift 4.0
- Xcode 9.0 and above
- Mac OS 10.13 and above
